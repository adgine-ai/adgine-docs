# Adgine Documentation

Welcome to **Adgine** — a SaaS platform for Generative Engine Optimization (GEO). This documentation helps you set up your brand, create content, analyze visibility, and connect integrations on [platform.adgine.ai](https://platform.adgine.ai).

## What can I do with Adgine?

| Capability | Description | Docs |
|------------|-------------|------|
| **Brand cognition** | AI analyzes your site and builds a brand profile | [Brand](/features/brand) |
| **Topics & prompts** | Organize GEO test topics and simulate AI queries | [Topics](/features/topics) |
| **Content** | AI-assisted outlines and articles with multi-platform publishing | [Content](/features/content) |
| **Opportunities** | Discover high-priority content optimization opportunities | [Opportunities](/features/opportunities) |
| **Visibility** | Track citations and rankings on AI platforms | [Analytics](/features/analytics) |
| **Traffic** | Monitor bot and human traffic | [Crawler](/features/crawler) |
| **Integrations** | Connect GA4, Cloudflare, WordPress, and more | [Integrations](/features/integrations/overview) |
