# Billing & Credits

Adgine uses subscriptions for feature access and credits for AI operations.

## Subscription

### Free trial

- **30-day free trial** on registration
- Basic features and initial credits included

### Paid plans

1. Go to **Subscribe** (`/subscribe`) or your profile
2. Compare plans and credit allowances
3. Pay via Stripe Checkout
4. Plan activates immediately

### Manage subscription

In Profile → **Subscription**:

- View current plan and expiry
- Upgrade or downgrade
- Manage payment methods via Stripe Customer Portal

## Credits

### What uses credits

| Operation | Uses credits |
|-----------|--------------|
| Brand cognition generation | Yes |
| Prompt generation | Yes |
| Content generation | Yes |
| Citation tests | Yes |
| AI refine / translate | Yes |

### View balance

- Top navigation shows **remaining credits**
- Profile → **Credits** for history

### Top up

1. Profile → **Buy credits**
2. Choose amount
3. Pay via Stripe — credits added instantly

## FAQ

### Out of credits?

Top up in your profile or upgrade to a plan with more credits.

### Payment methods?

Stripe supports major credit and debit cards. Availability varies by region.
