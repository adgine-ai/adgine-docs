# Quick Start

Understand Adgine's core workflow in 4 steps.

## Step 1: Create an account and project

1. Visit [platform.adgine.ai](https://platform.adgine.ai) to register
2. Use email, phone, or Google sign-in
3. Enter your **website domain** to create your first project

> No website yet? Choose subdomain quick-create or domain registration during project setup.

See [New user guide — Create project](/getting-started/onboarding-wizard#step-2-create-a-project).

## Step 2: Complete brand onboarding

After creating a project, the onboarding wizard guides you through:

1. **Brand cognition** — AI analyzes your site
2. **Region & language** — set your target market
3. **Topic selection** — choose or generate content topics
4. **Index status** — confirm search engine indexing

See [Brand cognition](/features/brand).

## Step 3: Configure topics and integrations

1. Add topics and generate GEO test prompts under **Manage → Topics**
2. Connect GA4, Cloudflare Worker, etc. under **Manage → Integrations**
3. Run citation tests to measure AI visibility

See [Topics](/features/topics) and [Integrations](/features/integrations/overview).

## Step 4: Create content and track results

1. Review opportunities under **Content → Opportunities**
2. Create content under **Content → Create**
3. Publish to WordPress or Adgine CMS
4. Track visibility under **Data → Analytics**

See [Content](/features/content) and [Analytics](/features/analytics).
