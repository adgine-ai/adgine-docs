# New User Guide

This guide walks you through the full onboarding path on the Adgine platform.

## Overview

```
Sign up / Sign in → Create project → Subscribe (if needed) → Brand wizard (4 steps) → Dashboard
```

## Step 1: Register and sign in

Visit [platform.adgine.ai](https://platform.adgine.ai):

| Method | Best for | Action |
|--------|----------|--------|
| **Email** | International users | Email → verification code → password |
| **Phone** | China users | Phone → SMS code → password |
| **Google** | Google account holders | One-click OAuth |

New accounts receive a **30-day free trial**.

## Step 2: Create a project

1. Enter your **website domain**
2. Optionally add a brand description
3. Click **Continue**

### No website?

- **Subdomain quick-create** — get a `yourbrand.adgine.net` site
- **Domain registration** — search and register a custom domain

## Step 3: Subscribe

If your trial expired or you need to upgrade:

1. Review plans on the subscribe page
2. Complete payment via Stripe
3. Return to the platform automatically

See [Billing & credits](/features/billing).

## Step 4: Brand onboarding wizard

Four steps at `/project/generate/:project_id`:

1. Brand cognition generation (2–5 min)
2. Region and language
3. Topic selection
4. Index status check

## Step 5: Enter Dashboard

After onboarding, access **Dashboard** for GA4 traffic, Cloudflare data, AI Agent KPIs, and quick links.

See [Dashboard](/features/dashboard).
