# 快速入門

用 4 步了解 Adgine 的核心工作流。

## 第 1 步：建立帳戶與專案

1. 訪問 [platform.adgine.ai](https://platform.adgine.ai) 註冊帳戶
2. 支援電子郵件、手機號或 Google 帳號註冊
3. 輸入您的**網站域名**建立第一個專案

## 第 2 步：完成品牌初始化

建立專案後，系統會引導您完成品牌初始化嚮導：

1. **品牌認知生成**
2. **地區與語言**
3. **主題選擇**
4. **索引狀態**

詳見 [品牌認知](/features/brand)。

## 第 3 步：配置主題與整合

1. 在 **管理 → 主題管理** 中添加主題並生成提示詞
2. 在 **管理 → 整合** 中連接 GA4、Cloudflare Worker 等
3. 執行引用測試

## 第 4 步：創作內容並追蹤效果

1. 在 **內容 → 機會** 中查看推薦方向
2. 在 **內容 → 創建內容** 中生成文章
3. 發布到 WordPress 或 Adgine CMS
4. 在 **數據 → 分析** 中追蹤表現
