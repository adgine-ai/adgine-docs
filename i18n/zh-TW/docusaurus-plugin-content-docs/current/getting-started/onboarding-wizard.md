# 新用戶嚮導

本文檔按平台實際流程，引導您從註冊帳戶到進入 Dashboard 的完整路徑。

## 流程概覽

```
註冊/登入 → 建立專案 → 訂閱（如需）→ 品牌初始化嚮導（4 步）→ Dashboard
```

## 步驟 1：註冊與登入

| 方式 | 適用場景 |
|------|---------|
| **電子郵件註冊** | 海外用戶 |
| **手機號註冊** | 國內用戶 |
| **Google 登入** | 已有 Google 帳號 |

註冊成功後，系統自動賦予 **30 天免費試用**。

## 步驟 2：建立專案

1. 輸入**網站域名**
2. （可選）填寫品牌介紹
3. 點擊 **繼續**

## 步驟 3：訂閱

若試用已過期，系統會引導至訂閱頁完成 Stripe 支付。

詳見 [訂閱與積分](/features/billing)。

## 步驟 4：品牌初始化嚮導

四步嚮導：品牌認知 → 地區語言 → 主題選擇 → 索引狀態。

## 步驟 5：進入 Dashboard

完成嚮導後進入 **Dashboard 總覽**。
