# Adgine 文檔

了解如何使用 Adgine 平台功能，提升品牌在 AI 搜尋中的可見性。

</div>

## 功能選擇說明

請先根據當前目標選擇功能：**首次使用**請閱讀 [快速入門](/getting-started/quick-start) 與 [新用戶嚮導](/getting-started/onboarding-wizard)；**配置品牌**請查看 [品牌認知](/features/brand)；**追蹤 AI 表現**請使用 [可見性分析](/features/analytics)；**創作內容**請進入 [內容創作](/features/content)；**連接數據源**請查看 [整合總覽](/features/integrations/overview)。

## 功能指南

<div class="feature-grid">

<a href="/docs/features/brand"><strong>品牌認知</strong><span>自動生成品牌畫像，作為後續主題、內容與引用測試的基礎。</span></a>

<a href="/docs/features/topics"><strong>主題與提示詞</strong><span>組織 GEO 測試主題，模擬用戶向 AI 引擎提問。</span></a>

<a href="/docs/features/content"><strong>內容創作</strong><span>從機會發現到文章發布，一站式 GEO 內容工作流。</span></a>

<a href="/docs/features/analytics"><strong>可見性分析</strong><span>追蹤品牌在 ChatGPT、Perplexity 等 AI 平台的表現。</span></a>

<a href="/docs/features/opportunities"><strong>機會分析</strong><span>發現值得優先推進的內容優化方向。</span></a>

<a href="/docs/features/crawler"><strong>爬蟲與流量</strong><span>監控 AI Bot 與人類訪問流量。</span></a>

<a href="/docs/features/dashboard"><strong>Dashboard 總覽</strong><span>查看專案核心指標與快捷入口。</span></a>

<a href="/docs/features/integrations/overview"><strong>第三方整合</strong><span>連接 GA4、Cloudflare、WordPress 與 Adgine CMS。</span></a>

</div>
