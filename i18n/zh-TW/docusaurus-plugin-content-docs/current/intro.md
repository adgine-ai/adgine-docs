# Adgine 產品文檔

歡迎使用 **Adgine** — 面向生成式引擎優化（GEO）的 SaaS 平台。本文檔幫助您在 [platform.adgine.ai](https://platform.adgine.ai) 上完成品牌設置、內容創作、可見性分析與第三方整合。

## 我能用 Adgine 做什麼？

| 能力 | 說明 | 文檔 |
|------|------|------|
| **品牌認知** | AI 自動分析您的網站，生成品牌畫像 | [品牌認知](/features/brand) |
| **主題與提示詞** | 組織 GEO 測試主題，模擬用戶向 AI 提問 | [主題與提示詞](/features/topics) |
| **內容創作** | AI 輔助生成大綱與文章，支援多平台發布 | [內容創作](/features/content) |
| **機會分析** | 發現值得優先推進的內容優化機會 | [機會分析](/features/opportunities) |
| **可見性分析** | 追蹤品牌在 AI 平台的引用與排名 | [可見性分析](/features/analytics) |
| **流量監控** | 查看 Bot 與人類訪問流量 | [爬蟲與流量](/features/crawler) |
| **第三方整合** | 連接 GA4、Cloudflare、WordPress 等 | [整合總覽](/features/integrations/overview) |
