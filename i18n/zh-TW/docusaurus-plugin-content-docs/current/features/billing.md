# 訂閱與積分

Adgine 採用訂閱制 + 積分消耗的模式。

## 訂閱套餐

### 免費試用

- 註冊即獲得 **30 天免費試用**

### 付費套餐

1. 進入 **訂閱** 頁面查看套餐
2. 選擇後跳轉 Stripe 完成支付
3. 支付成功後自動激活

### 管理訂閱

在個人中心 → **訂閱** 中管理套餐與支付方式。

## 積分系統

積分用於品牌認知生成、提示詞生成、內容生成、引用測試等 AI 操作。

在頂部導航欄或個人中心查看 **剩餘積分** 與消耗記錄。

## 常見問題

### 積分用完了怎麼辦？

可在個人中心充值積分，或升級到包含更多積分的套餐。
