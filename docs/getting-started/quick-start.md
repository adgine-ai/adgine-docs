# 快速入门

用 4 步了解 Adgine 的核心工作流。

## 第 1 步：创建账户与项目

1. 访问 [platform.adgine.ai](https://platform.adgine.ai) 注册账户
2. 支持邮箱、手机号或 Google 账号注册
3. 注册成功后，输入您的**网站域名**创建第一个项目

> 还没有网站？可在创建项目时选择「使用二级域名快速创建网站」或「注册域名创建网站」。

详见 [新用户向导 — 创建项目](/getting-started/onboarding-wizard#步骤-2创建项目)。

## 第 2 步：完成品牌初始化

创建项目后，系统会引导您完成品牌初始化向导：

1. **品牌认知生成** — AI 分析您的网站，生成品牌介绍、理想客户、竞品与品牌视角
2. **地区与语言** — 设置目标市场
3. **主题选择** — 选择或生成内容主题
4. **索引状态** — 确认网站可被搜索引擎索引

完成后即可进入 Dashboard。详见 [品牌认知](/features/brand)。

## 第 3 步：配置主题与集成

1. 在 **管理 → 主题管理** 中添加主题，并为每个主题生成 GEO 测试提示词
2. 在 **管理 → 集成** 中连接 GA4、Cloudflare Worker 等数据源
3. 运行引用测试，了解品牌在 AI 平台的表现

详见 [主题与提示词](/features/topics) 与 [集成总览](/features/integrations/overview)。

## 第 4 步：创作内容并追踪效果

1. 在 **内容 → 机会** 中查看推荐的内容优化方向
2. 在 **内容 → 创建内容** 中生成大纲与文章
3. 发布到 WordPress 或 Adgine CMS
4. 在 **数据 → 分析** 中追踪可见性与引用表现

详见 [内容创作](/features/content) 与 [可见性分析](/features/analytics)。
