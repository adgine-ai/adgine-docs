# 新用户向导

本文档按平台实际流程，引导您从注册账户到进入 Dashboard 的完整路径。

## 流程概览

```
注册/登录 → 创建项目 → 订阅（如需）→ 品牌初始化向导（4 步）→ Dashboard
```

## 步骤 1：注册与登录

访问 [platform.adgine.ai](https://platform.adgine.ai)，根据您的地区选择注册方式：

| 方式            | 适用场景         | 操作                     |
| ------------- | ------------ | ---------------------- |
| **邮箱注册**      | 海外用户         | 填写邮箱 → 收取验证码 → 设置密码    |
| **手机号注册**     | 国内用户         | 填写手机号 → 收取短信验证码 → 设置密码 |
| **Google 登录** | 已有 Google 账号 | 点击 Google 登录按钮一键授权     |

注册成功后，系统自动赋予 **30 天免费试用**。并赠送credits积分。

:::info API Key
若您需要在 OpenClaw 或 Hermes Agent 等平台使用 Adgine Skills，可在个人中心 [创建 API Key](/features/api-keys)。
:::

## 步骤 2：创建项目

登录后，若您尚未选择项目，系统会引导至项目创建页：

1. 输入您的**网站域名**（如 `example.com`）
2. （可选）填写品牌介绍，帮助 AI 更准确地分析
3. 点击 **继续** 创建项目

### 没有网站？

* **使用二级域名快速创建网站**：获得 `yourbrand.adgine.net` 子域名站点
* **注册域名创建网站**：搜索并注册专属域名，自动建站

## 步骤 3：订阅

若免费试用已过期或需要升级套餐，系统会引导至订阅页：

1. 查看可用套餐与功能对比
2. 选择套餐后跳转 Stripe 完成支付
3. 支付成功后自动返回平台

详见 [订阅与积分](/features/billing)。

## 步骤 4：品牌初始化向导

创建项目后（且订阅有效），进入 **品牌初始化向导**（`/project/generate/:project_id`），共 4 步：

### 4.1 品牌认知生成

* 系统分析您的网站内容，自动生成品牌画像
* 包含：品牌介绍、理想客户、竞争对手、品牌视角
* 生成过程约 2–5 分钟，可查看实时进度

### 4.2 地区与语言

* 设置目标市场地区与内容语言
* 影响后续提示词生成与内容创作的默认语言

### 4.3 主题选择

* 系统根据品牌认知推荐内容主题
* 可选择已有主题或让 AI 生成新主题

### 4.4 索引状态

* 确认网站可被搜索引擎与 AI 爬虫索引
* 若未就绪，按提示完成 Cloudflare 或 SEO 配置

## 步骤 5：进入 Dashboard

完成向导后，您将进入 **Dashboard 总览**：

* 查看 GA4 流量、Cloudflare Worker 数据、AI Agent KPI
* 通过快捷入口进入分析、内容、管理等模块

详见 [Dashboard 总览](/features/dashboard)。

## 常见问题

### 向导中途退出，如何继续？

重新登录并选择项目，若品牌初始化未完成，系统会自动引导您回到向导。

### 可以跳过品牌初始化吗？

品牌认知是后续主题生成、内容创作与引用测试的基础，建议完成全部步骤。

### 如何切换项目？

点击顶部导航栏的 **切换项目**，选择或创建其他项目。每个项目独立管理品牌、主题与内容。
