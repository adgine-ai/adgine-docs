# 机会分析

机会分析帮助您发现值得优先推进的内容优化方向，基于品牌数据、竞品信号与市场趋势进行智能评分。

## 适用场景

- 不确定下一步该创作什么内容
- 希望基于数据驱动选择内容优先级
- 发现实时热点或内容缺口

## 功能概览

进入 **内容 → 机会**（`/geo/opportunities`），查看系统为您的项目生成的机会列表。

每条机会包含：

| 字段 | 说明 |
|------|------|
| **标题** | 机会主题 |
| **分类** | 实时热点、前瞻事件、内容缺口、社区参与等 |
| **总分** | 综合评分（相关性、流量潜力、竞争度、紧迫性、AI 引用概率） |
| **机会原因** | 为什么这是一个值得关注的方向 |

## 操作步骤

### 查看机会列表

1. 确保已选择正确的项目
2. 进入机会页面，等待数据生成完成
3. 按评分排序，优先关注高分机会

### 查看机会详情

点击任意机会，查看：

- **评分雷达** — 五个维度的详细得分
- **内容覆盖度** — 您是否已有相关内容
- **执行指南** — 建议的执行步骤
- **信号来源** — 数据来源与参考链接

### 从机会创建内容

1. 在机会详情页点击 **创建内容**
2. 系统自动带入机会主题与执行指南
3. 进入内容创作流程

## 机会分类说明

| 分类 | 说明 |
|------|------|
| **实时热点** | 当前市场或行业热点话题 |
| **前瞻事件** | 即将到来的行业事件或趋势 |
| **内容缺口** | 竞品已覆盖但您尚未涉及的话题 |
| **社区参与** | 社区讨论中与您品牌相关的机会 |

:::info Beta 功能
机会分析目前处于 Beta 阶段，数据会持续优化。首次生成可能需要一些时间。
:::

## 常见问题

### 为什么没有机会数据？

机会分析依赖品牌认知与集成数据。请确保已完成品牌初始化，并连接 GA4 等数据源。

### 机会多久更新一次？

系统定期重新分析。具体频率取决于您的订阅套餐与数据同步状态。
