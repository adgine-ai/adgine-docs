# 品牌认知

品牌认知是 Adgine 的核心基础。系统通过分析您的网站，自动生成 AI 可理解的品牌画像，用于后续的主题生成、内容创作与引用测试。

## 适用场景

- 刚创建项目，需要建立品牌档案
- 品牌定位发生变化，需要更新认知内容
- 内容创作前，确保 AI 了解您的品牌调性与目标受众

## 品牌认知包含什么？

| 栏目 | 说明 |
|------|------|
| **品牌介绍** | 品牌使命、核心价值与差异化特点 |
| **理想客户** | 目标受众画像、痛点与决策因素 |
| **竞争对手** | 直接竞品与间接竞品分析 |
| **品牌视角** | 品牌在行业中的独特主张与立场 |

## 操作步骤

### 首次生成（品牌初始化向导）

1. 创建项目后，系统自动进入品牌初始化向导
2. 第一步即为品牌认知生成，等待 AI 分析完成（约 2–5 分钟）
3. 生成完成后可预览四栏内容，确认无误后进入下一步

### 在品牌中心查看与编辑

1. 进入 **管理 → 品牌中心**（`/geo/brand`）
2. 查看当前品牌认知的四栏内容
3. 点击任意栏目进行手动编辑
4. 保存后即时生效

### 重新生成

若网站内容有重大更新，可触发重新生成：

1. 在品牌中心点击 **重新生成**
2. 选择目标语言与地区
3. 等待后台任务完成（可通过任务列表查看进度）

:::warning 注意
重新生成会覆盖当前品牌认知内容。若有手动编辑的内容，建议先备份重要段落。
:::

## 生成原理（简要）

1. **网站内容提取** — 抓取主页与 About 页面
2. **外部信息补充** — 搜索竞品与用户评价，丰富认知数据
3. **四栏内容生成** — AI 基于分析结果生成各栏目文案
4. **质量检查** — 自动校验语言、字数与内容一致性

## 常见问题

### 生成失败怎么办？

检查网站是否可正常访问，域名是否正确。若持续失败，可在品牌中心查看任务详情并重试。

### 可以只编辑部分内容吗？

可以。四栏内容均可独立编辑，无需重新生成全部。

### 品牌认知与主题生成的关系？

主题与提示词的 AI 生成依赖品牌认知中的中间数据。请先完成品牌认知，再生成主题提示词。
