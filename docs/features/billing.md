# 订阅与积分

Adgine 采用订阅制 + 积分消耗的模式。订阅决定功能权限，积分用于 AI 相关操作。

## 订阅套餐

### 免费试用

- 注册即获得 **30 天免费试用**
- 包含基础功能与一定积分额度

### 付费套餐

1. 进入 **订阅** 页面（`/subscribe`）或在个人中心查看
2. 对比各套餐的功能与积分额度
3. 选择套餐后跳转 Stripe 完成支付
4. 支付成功后自动激活

### 管理订阅

在个人中心 → **订阅** 中：

- 查看当前套餐与到期时间
- 升级或降级套餐
- 通过 Stripe Customer Portal 管理支付方式

## 积分系统

### 积分用途

| 操作 | 消耗积分 |
|------|---------|
| 品牌认知生成 | 是 |
| 提示词生成 | 是 |
| 内容大纲/文章生成 | 是 |
| 引用测试 | 是 |
| AI 精修/翻译 | 是 |

### 查看积分余额

- 顶部导航栏显示 **剩余积分**
- 个人中心 → **积分** 查看详细余额与消耗记录

### 充值积分

1. 进入个人中心 → **积分充值**
2. 选择充值金额
3. 跳转 Stripe 完成支付
4. 积分即时到账

### 积分账单

在个人中心查看积分消耗历史，包含每次操作的类型、时间与消耗量。

## 操作步骤

### 订阅新套餐

1. 访问 `/subscribe` 或在试用到期时被引导
2. 选择套餐 → Stripe Checkout → 支付
3. 成功后返回平台，功能立即解锁

### 取消订阅

1. 进入 Stripe Customer Portal（通过个人中心链接）
2. 取消自动续费
3. 当前周期内功能仍可用，到期后降级

## 常见问题

### 积分用完了怎么办？

可在个人中心充值积分，或升级到包含更多积分的套餐。

### 支持哪些支付方式？

通过 Stripe 支持主流信用卡与借记卡。具体可用方式取决于您的地区。

### 可以开具发票吗？

联系客服提供开票信息，或在 Stripe Portal 中下载收据。
