# Adgine 产品文档

欢迎使用 **Adgine** — 面向生成式引擎优化（GEO）的 SaaS 平台。本文档帮助您了解如何在 [platform.adgine.ai](https://platform.adgine.ai) 上完成品牌设置、内容创作、可见性分析与第三方集成。

## 我能用 Adgine 做什么？

| 能力 | 说明 | 文档 |
|------|------|------|
| **品牌认知** | AI 自动分析您的网站，生成品牌画像 | [品牌认知](/features/brand) |
| **主题与提示词** | 组织 GEO 测试主题，模拟用户向 AI 提问 | [主题与提示词](/features/topics) |
| **内容创作** | AI 辅助生成大纲与文章，支持多平台发布 | [内容创作](/features/content) |
| **机会分析** | 发现值得优先推进的内容优化机会 | [机会分析](/features/opportunities) |
| **可见性分析** | 追踪品牌在 AI 平台的引用与排名 | [可见性分析](/features/analytics) |
| **流量监控** | 查看 Bot 与人类访问流量 | [爬虫与流量](/features/crawler) |
| **第三方集成** | 连接 GA4、Cloudflare、WordPress 等 | [集成总览](/features/integrations/overview) |
