# Adgine 文档

了解如何使用 Adgine 平台功能，提升品牌在 AI 搜索中的可见性。

</div>

## 功能选择说明

请先根据当前目标选择功能：**首次使用**请阅读 [快速入门](/getting-started/quick-start) 与 [新用户向导](/getting-started/onboarding-wizard)；**配置品牌**请查看 [品牌认知](/features/brand)；**追踪 AI 表现**请使用 [可见性分析](/features/analytics)；**创作内容**请进入 [内容创作](/features/content)；**连接数据源**请查看 [集成总览](/features/integrations/overview)。

## 功能指南

<div class="feature-grid">

<a href="/docs/features/brand"><strong>品牌认知</strong><span>自动生成品牌画像，作为后续主题、内容与引用测试的基础。</span></a>

<a href="/docs/features/topics"><strong>主题与提示词</strong><span>组织 GEO 测试主题，模拟用户向 AI 引擎提问。</span></a>

<a href="/docs/features/content"><strong>内容创作</strong><span>从机会发现到文章发布，一站式 GEO 内容工作流。</span></a>

<a href="/docs/features/analytics"><strong>可见性分析</strong><span>追踪品牌在 ChatGPT、Perplexity 等 AI 平台的表现。</span></a>

<a href="/docs/features/opportunities"><strong>机会分析</strong><span>发现值得优先推进的内容优化方向。</span></a>

<a href="/docs/features/crawler"><strong>爬虫与流量</strong><span>监控 AI Bot 与人类访问流量。</span></a>

<a href="/docs/features/dashboard"><strong>Dashboard 总览</strong><span>查看项目核心指标与快捷入口。</span></a>

<a href="/docs/features/integrations/overview"><strong>第三方集成</strong><span>连接 GA4、Cloudflare、WordPress 与 Adgine CMS。</span></a>

</div>
